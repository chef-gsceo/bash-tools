My tools & utilities for Linux 


## Installation

Clone the repo:

git clone https://github.com/chef-gsceo/bash-scripts.git

Add to PATH:

export PATH="$HOME/bash-scripts/bin:$PATH"


## Utilities

### pcctl

Remote PC management tool.

Features:
- Wake-on-LAN
- SSH connection
- RSYNC server-client

You need to set your own values for the variables according to your system settings for correct work

### fnd

Grep wrapper with additional checks.

### ff

Debug version of fnd

### Monitor

Tiny Uptime & Sensors wrapper
