My CLI Utilities for Linux


## Installation

Clone the repo:

git clone https://github.com/chef-gsceo/bash-scripts.git

Add to PATH:

export PATH="$HOME/bash-scripts/bin:$PATH"


## Utilities

### pcctl

Remote PC management tool.

Features:
- Wake-on-LAN
- SSH connection
- RSYNC server-client

### fnd

Grep wrapper with additional checks.

### ff

debug version of fnd
