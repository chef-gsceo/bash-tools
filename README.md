My CLI Utilities for Linux


## Installation

Clone the repo:

git clone https://github.com/chef-gsceo/bash-scripts.git

Add to PATH:

export PATH="$HOME/bash-scripts/bin:$PATH"


## Utilities

### pcctl

Remote PC management tool.

Features:
- Wake-on-LAN
- SSH connection
- File transfer via rsync

### fnd

Wrapper around grep with additional checks.

### ff

debug version of fnd


## Notes

This is a personal toolkit, constantly evolving.
Scripts are designed for Linux environments.
